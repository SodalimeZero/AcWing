## 题目：区间选点
给定 $N$ 个闭区间 [ $a_i$, $b_i$]，请你在数轴上选择尽量少的点，使得每个区间内至少包含一个选出的点。

输出选择的点的最小数量。

位于区间端点上的点也算作区间内。

## 输入格式
第一行包含整数 $N$ ，表示区间数。接下来 $N$ 行，每行包含两个整数 $a_i$、 $b_i$，表示一个区间的两个端点。

## 输出格式
输出一个整数，表示所需的点的最小数量。

## 数据范围

$1≤N≤10^5,$
$−10^9≤a_i≤b_i≤10^9$

**输入样例**
```r
3
-1  1
2   4
3   5
```
**输出样例**
```
2
```

## 解题思路
* 将每个区间按照右端点从小到大排序
* 从前往后依次枚举每个区间
   * 如果当前区间中已经包含该点，则直接pass
   * 否则，选择当前区间的最右侧端点

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

const int N = 100010;

struct Range
{
    int l, r;
    bool operator< (const Range &W)const
    {
        return r < W.r;
    }
}range[N];

int main() {
    int n;
    cin >> n;
    
    for (int i = 0; i < n; i ++ ) {
        scanf("%d%d", &range[i].l, &range[i].r);
    }

    sort(range, range + n);
    
    int numOfNodes = 1;
    int current = range[0].r; // 第一个值为最小的右侧端点值
    for (int i = 1; i < n; i++) {
        while (range[i].l <= current && i < n) {
            i++;
        } 
        if (i < n) {
            numOfNodes++;
            current = range[i].r;
        }
    }
    cout << numOfNodes << endl;
    return 0;
}
```
